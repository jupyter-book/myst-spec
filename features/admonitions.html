Admonitions¶

Admonitions highlight a particular block of text that exists slightly apart from the narrative of your page, like so:

Note

Here is a note!

Specification¶

Admonition node for drawing attention to text, separate from the neighboring content

  • type*: string (“admonition”) - See Node

  • kind: string (“attention” | “caution” | “danger” | “error” | “hint” | “important” | “note” | “seealso” | “tip” | “warning”) - kind of admonition, to determine styling

  • class: string - admonition class info to override kind

  • children: array (AdmonitionTitle | FlowContent)

  • position: object (Position) - See Node

  • data: object - See Node

Example¶

type: root
children:
  - type: directive
    kind: note
    args: This is a title in myst
    value: and an example of a note admonition.
    children:
      - type: admonition
        kind: note
        children:
          - type: admonitionTitle
            children:
              - type: text
                value: This is a title in myst
          - type: paragraph
            children:
              - type: text
                value: and an example of a note admonition.
