Tables¶

Specification¶

Two-dimensional table data

  • type*: string (“table”) - See Node

  • align: string (“left” | “center” | “right”)

  • children*: array (TableRow) - See Parent

  • position: object (Position) - See Node

  • data: object - See Node

Example¶

| header 1 | header 2 |
|:---|---:|
| 3 | 4 |

type: root
children:
  - type: table
    children:
      - type: tableRow
        children:
          - type: tableCell
            header: true
            align: left
            children:
              - type: text
                value: header 1
          - type: tableCell
            header: true
            align: right
            children:
              - type: text
                value: header 2
      - type: tableRow
        children:
          - type: tableCell
            align: left
            children:
              - type: text
                value: '3'
          - type: tableCell
            align: right
            children:
              - type: text
                value: '4'
